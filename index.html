Tutorial Modifikasi Grok Beta

Langkah-Langkah Modifikasi

  1. Masuk ke MT Manager

    Unduh dan instal aplikasi MT Manager atau gunakan versi modifikasi jika belum memilikinya.

  2. Ekstrak Aplikasi

    Buka sidebar MT Manager, klik "Ekstrak", pilih aplikasi Grok, lalu ubah menjadi APK menggunakan SplitApks atau langsung di MT Manager.

  3. Akses Fungsi Kill Sign

    Klik "Fungsi", cari "Kill Sign", masuk ke bagian DEX, pilih semua, cari string "grok_pro", klik navigasi, pilih Lgrok_api_v2/SubscriptionTier;, tekan "Find Usage", dan cari file smali f.

  4. Bypass Subscription

    Edit kode smali seperti berikut:

    .method public final c()Z
    .registers 1
    const/4 p0, 0x1
    return p0
.end method

.method public final d()Z
    .registers 1
    const/4 p0, 0x1
    return p0
.end method
  5. Simpan dan Tanda Tangan

    Simpan perubahan, tanda tangani APK, masuk ke bagian manifest, hapus activity pairip. Selamat, Anda berhasil memodifikasi aplikasi Grok!
